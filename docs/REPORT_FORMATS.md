# KINTYRE DAM Report Formats

**Version:** 0.2

---

# Purpose

This document defines every report generated by KINTYRE DAM.

Report formats are considered part of the public interface between project
components.

Existing columns should remain stable.

New columns may be appended in future versions.

# Compatibility Policy

Report schemas are versioned.

Columns should never be renamed once released.

Applications consuming these reports should ignore unknown columns to remain
compatible with future releases.
